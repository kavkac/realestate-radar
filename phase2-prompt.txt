## TASK 2: Zemljisce lookup (add to lib/gurs-api.ts)

Add VRSTA_RABE map and ParcelaData interface:

export interface ParcelaData {
  parcelnaStevila: string;
  povrsina: number | null;
  vrstaRabe: string | null;
  boniteta: number | null;
  katastrskiRazred: number | null;
  katastrskiDohodek: number | null;
}

Add function getParcele(koId: number, stStavbe: number): Promise<ParcelaData[]>

The PARCELE_TABELA WFS is at BASE_KN (https://ipi.eprostor.gov.si/wfs-si-gurs-kn-osnovni/wfs)
Layer: SI.GURS.KN:PARCELE_TABELA

To link building to parcels:
1. First fetch STAVBE_TABELA with filter: KO_ID={koId} AND ST_STAVBE={stStavbe}
2. Check if the feature has a PARCELA or ST_PARCELE field
3. If yes, use that to fetch PARCELE_TABELA with: KO_ID={koId} AND ST_PARCELE='{value}'
4. Otherwise try fetching PARCELE_TABELA with: KO_ID={koId} AND ST_STAVBE={stStavbe}
5. Parse fields: ST_PARCELE, POVRSINA, VRSTA_RABE_ID, BONITETA_TALA, KATASTRSKI_RAZRED, KATASTRSKI_DOHODEK
6. Return array (may be empty - that's fine)

VRSTA_RABE map (common values): 1=Njiva, 2=Vrt, 6=Travnik, 21=Gozd, 30=Pozidano stavisca, 31=Cesta, 39=Park, 41=Ostalo
